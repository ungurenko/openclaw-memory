# Лаборатория улучшений (Improvement Plane)

Здесь хранятся эксперименты по улучшению логики, промптов и навыков Краба.

## Структура
- `/hypotheses` — черновики изменений и их обоснование.
- `/tests` — результаты прогонов Golden Set для новых версий.
- `performance-metrics.json` — история оценок лени и качества.

## Текущие цели
1. Снизить "Уровень лени" до < 3 к концу недели.
2. Автоматизировать проверку Golden Set через скрипт.
