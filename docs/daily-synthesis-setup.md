# Daily Synthesis — Промпт для cron-задачи

**Задача:** Вечерний обзор дня с автообновлением памяти

**Расписание:** Ежедневно 21:00 (Мск)

**Промпт:**

```
Daily Synthesis — вечерний обзор дня.

1. Прочитай:
   - memory/active-tasks.md
   - memory/recent-context.md
   - memory/YYYY-MM-DD.md (сегодняшний дневник)

2. Обнови файлы:
   - active-tasks.md: что выполнено → архивировать, что осталось → оставить, новое → добавить
   - recent-context.md: актуализировать резюме последних 2-3 дней
   - Если есть значимые выводы/решения → добавить в MEMORY.md

3. Отправь Александру краткий обзор (3-7 строк):
   - Что сделано за день
   - Что на завтра / ближайшие дни
   - Если есть незавершённое → напомнить

4. Если видишь явное улучшение процесса (1 конкретная идея) → предложить

Правила:
- Учитывать тихие часы (23:00-07:00) — если 21:00 попадает в тихие, пропустить
- Если день был пустой (ничего не делали) → отправить только если есть важные напоминания
- Формат: структурированно, с эмодзи, коротко
```

---

## Создание cron-задачи:

После возвращения Александра выполнить:

```bash
openclaw cron add
```
