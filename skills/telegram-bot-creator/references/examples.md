# Примеры использования Telegram Bot Creator

## Пример 1: Бот для улучшения промптов

**Запрос пользователя:**
```
хочу сделать телеграм-бота который будет улучшать промпт, я присылаю ему простой промпт а он его доделывает нужно встроить туда ИИ, через openrouter, хочу использовать модель deepseek/deepseek-v3.2 создай такого бота и разверни его на railway
```

**Входные данные:**
- Telegram Bot Token: `8560371121:AAFmozc7xP44cte-FU49SFjXrG3EhGMIusE`
- OpenRouter API Key: `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`
- Имя бота: `PromptImproverBot`
- Модель: `deepseek/deepseek-v3.2`

**Настройка SYSTEM_PROMPT:**
```
Ты — эксперт по созданию промптов для ИИ-моделей. Твоя задача — улучшать промпты пользователей.

Правила улучшения:
1. Добавь контекст и роль для ИИ
2. Уточни желаемый формат ответа
3. Добавь примеры если уместно
4. Сделай промпт структурированным и понятным
5. Сохрани исходную intent пользователя
6. Отвечай на том же языке, что и исходный промпт

ВАЖНО: Возвращай ТОЛЬКО улучшенный промпт, без объяснений и комментариев.
```

**Результат:**
- Бот: @Promp_AItImprover_Bot
- Railway: https://railway.com/project/3cb6bf7e-358e-4ea4-b7cd-4074f2bae8fb
- Webhook: https://prompt-improver-bot-production.up.railway.app/telegram-webhook

## Пример 3: Создание бота с другим функционалом

**Запрос:**
```
Создай бот-переводчик с русского на английский
- Token: <токен>
- OpenRouter Key: <ключ>
- Имя: TranslateBot
```

**Настройка SYSTEM_PROMPT:**
```
Ты — профессиональный переводчик с русского на английский.
Переводи текст точно и естественно, сохраняя стиль оригинала.
Возвращай ТОЛЬКО перевод без объяснений.
```

## Типичные модели OpenRouter

**Бесплатные:**
- `arcee-ai/trinity-large-preview:free` - быстрая, качественная
- `stepfun/step-3.5-flash:free` - очень быстрая, базовое качество
- `google/gemma-3-1b-it:free` - лёгкая модель Google

**Платные (быстрые):**
- `openrouter/aurora-alpha` - быстрая, хорошее качество
- `deepseek/deepseek-chat` - быстрая, качественная

**Платные (качественные):**
- `anthropic/claude-3.5-sonnet` - очень качественная
- `openai/gpt-4-turbo` - высокое качество
